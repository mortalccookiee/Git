# Git

这个仓库用于记录学习 GitHub 和 Git 使用过程中的练习代码、笔记和小脚本，也作为早期代码上传测试仓库。

## 主要内容

- `hello_vscode.py`：第一次在 VS Code 中编写 Python 代码的示例。
- `reproject_rasters.py`：批量读取 `.tif` 栅格文件，并按参考栅格的坐标系进行重投影。
- `resample_rasters_30m.py`：批量将 `.tif` 栅格文件重采样到 30 米分辨率。
- `index.txt`：早期 GitHub 使用记录。
- `my-quarto-project/`：Quarto 学习示例项目。

## 回顾时怎么看

这个仓库不是正式项目仓库，主要用于回顾自己刚开始使用 GitHub、VS Code、Quarto 和 Python 脚本上传的过程。

## 文件名整理记录

为避免中文文件名在 GitHub、URL、命令行或部分工具中显示为乱码、编码串或横线，已将部分脚本文件改为 ASCII 英文文件名：

- `1.py` -> `hello_vscode.py`
- `投影变换.py` -> `reproject_rasters.py`
- `重采样30m.py` -> `resample_rasters_30m.py`
